# How to Add Action to the FoldersView

*content to be added*

To see action in foldersView context-menu it should be added to foldersView configuration.

Example for Rename Folder action:

```xml
<FoldersView id="path">
    <Actions>
        <Action ID="rename_folder"/>         
        <!-- other actions -->
    </Actions>
    <!-- rest config-->
</FoldersView>
```
